# HCI-Academic-Writing
HCI-Academic-Writing: a Claude Skill built from real thesis reviewer feedback + the HCI literature (Wobbrock &amp; Kientz, Olsen, Swales, Phillips &amp; Pugh). Catches defensive writing, fixes contribution framing &amp; novelty claims, calibrates Master's vs PhD standards. For students writing proposals, theses, and papers Claude will actually help you !
